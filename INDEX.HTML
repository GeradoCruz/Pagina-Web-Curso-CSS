PRACTICA 1

La sintaxis basica de CSS

selector{
propiedad:valor;
}
selector:Es el elemento o los elementos que vamos a cambiarle la propiedad
Propiedad:que es lo que deseamos cambiar como el ancho,el color,la posicion
el tamaño,agregamos bordes etc

Tipos de Selectores

Universal:Selecciona todos los elementos y cambia la forma de color
*{
color;red
}
De tipo:los selectores de tipo son los que seleccionamos por elementos
h1{
color;red
}

Clase: Agregar el atributo
class="titulo-h2"
.titulo-h2
solo cambiamos el color de un solo elemento
ID: Para seleccionar un elemento se utiliza el
#id-elemnt{
color:red
}
con este selector se identifica un elemento unico, para poder trabjar de la forma coreecta
no se pueden repetir es incorrecto utilizar un mismo id para muchos elementos
Por atributo: Para utilizar este selector se utiliza
[real]="sii"{
color greed
}

puede ser cualquier atributo
Descendiente:Se aplica a los elementos que estan dentro
de otro elemento (o mas exactamente, un elemento que es descendiente
de otro elemento).Por ejemplo,una lista desordenada tiene una etiqueta con
descendientes
h2 p {
color:red;
}
Pseudo-clases:Especidica un estado especial del elemento seleccionado.Por ejemplo:hover
aplicara un estilo cuando el usuario haga hover sobre el elemento especificado por el
selector.
b:hover{
color:red;
}

Medidas

Medidas Relativas

Dependen de algo como las variables

Medidad Fijas

Son las que tiene una unidad de medida, como lo pixeles, centimetros, milimetros, puntos, etc.

Cambio de valor em
.contact-form{
font-size: 25px;
}
.contact-form__h2{
font-size: 1em;
}
1em se hereda a 25px por la estructura de cambio
Además tenemos tambien los vh y vw que son muy utiles, vh indica la medida en ancho, ejemplo 100vh, ocupará todo el ancho de nuestro contenedor o pantalla y 100vw ocupara toda la pantalla o contenedor en largo.

Normalize

Normalmente todos los estilos que se colocan dentro de la pagina cuando escribimos estructura html, el navegador lo coloca por defecto, es decir, todos los margenes, paddings, anchos, y alturas, etc. Es por ello que Normalize hace es reiniciar los estilos para que nosotros podamos trabajar comodamente, para ello basta con descargar normalize y enlazarlo a nuestra pagina, o tambien se puede con un enlace externo para no descargar.

Teoria de cajas

Cajas en linea.
Lo elementos en linea ocupan solo lo devido del contenido que se le aplica.
Y cajas en bloque.
Los elementos en bloque ocupan todo el tamaño de la pagina. Y todos los elementos que vienen despues van a ir abajo.
Pero hay una propiedad que le podemos colocar para que sea en linea los elementos en bloque y viceversa.
Esta propiedad es display:inline o display:block y de esta manera podemos cambiarle sus propiedades.
